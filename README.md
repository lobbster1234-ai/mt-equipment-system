# MT 部門設備借用管理系統

🦐 一個簡潔實用的設備追蹤管理工具

## 功能特色

### 📦 設備登記
- 完整資產資訊記錄（編號、類型、描述、數量）
- 設定保管人與所屬部門
- 記錄存放位置與購入日期
- 初始狀態設定（可借用/維修中）

### 📤 借用管理
- 一鍵借用設備
- 記錄借用人、部門、時間
- 設定預計歸還日期
- 添加借用備註

### 📥 歸還管理
- 快速歸還設備
- 自動更新設備狀態
- 保留完整借用歷史

### 🔍 搜尋與篩選
- 關鍵字搜尋（設備名稱、編號、保管人）
- 依部門篩選
- 依狀態篩選（可借用/已借出/維修中）

## 使用方式

### 本地開啟
```bash
# 在瀏覽器開啟
open mt-equipment-system/index.html

# 或用 Python 啟動本地伺服器
cd mt-equipment-system
python3 -m http.server 8080
# 然後訪問 http://localhost:8080
```

### 資料儲存
- 使用瀏覽器 LocalStorage 儲存
- 資料保存在用戶瀏覽器中
- 無需後端伺服器

## 資料欄位對照

| 欄位 | 說明 | 範例 |
|------|------|------|
| fix_no | 設備編號 | MT-2024-001 |
| fix_type | 設備類型 | 儀器/工具/電腦設備/測試設備 |
| descrip | 設備描述 | 示波器 Tektronix TDS2014C |
| qty_asset | 資產數量 | 1 |
| location | 存放位置 | A 棟 3F 實驗室 |
| dept_id | 所屬部門 | MT/EE/ME/其他 |
| keeper | 保管人 | 王小明 |
| dt_purchase | 購入日期 | 2024-01-15 |
| status | 狀態 | available/borrowed/maintenance |

## 技術架構

- **純前端**：HTML5 + CSS3 + Vanilla JavaScript
- **無需依賴**：不需要任何框架或庫
- **響應式設計**：支援桌面與行動裝置
- **LocalStorage**：資料持久化在瀏覽器
